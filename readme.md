# i-work-2026

这是 apple 当前的工作/技术笔记仓库，主要用来沉淀日常问题、环境搭建记录和常用命令。

## 当前内容方向

### 🌅 work logs
按时间记录工作中的问题、处理过程和临时结论，方便后面回查。

### 🖥️ program
偏开发与运维问题整理，目前覆盖这些主题：

- **Nginx**
  - 部署时如何设置文件缓存过期时间
- **Docker**
  - Linux 系统如何安装 Docker
  - 如何删除没用的镜像
- **宝塔 / aaPanel**
  - Linux 如何安装宝塔
- **Kubernetes / k8s**
  - pods 挂掉后的日志查询
- **Git**
  - 如何连接远程仓库
  - 如何生成 SSH Key
- **Go**
  - Ubuntu 如何安装 Go
  - Ubuntu 如何安装 gf
  - Ubuntu 如何删除 Go
  - mac 如何安装 Go
- **Linux**
  - 服务器如何禁用密码登录
  - 服务器如何配置免密登录
- **MySQL**
  - mac 如何安装 MySQL
  - 如何查看 MySQL 服务是否运行
  - 如何把 SQL 文件导入远程服务器
  - 如何清空一张表
  - 如何登录远程 MySQL

### 🌈 电脑环境安装
记录电脑和开发环境初始化相关内容，方便新机器或重装后快速恢复。

## 适合后续继续补充的内容

- 问题

- 常见报错
